LedSwitcher


The assignment calls for a program which can:

Take data input from a text file/url.
	Method1: read a file line by line.

Parse that data into a data structure.
	Method2: add the data in each line to a two dimentional list.

Apply that data to a template data structure.
	Method3: get the size of the array specified in the input file.
	Method4: create a template data structure of the correct size.
	Method5: apply the list taken from the file to the template list.

Return a interger query of the resulting datastructure.
	Method6: read the number of items in the list which are in an on state.
